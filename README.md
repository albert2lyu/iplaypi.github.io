# 安装须知


如果本地环境缺失 `hexo` 环境，可以安装：

```
npm --registry=https://registry.npm.taobao.org install hexo --save
```

注意，如果使用官方的镜像源 `https://registry.npmjs.org` 而不设置代理的话，由于网络原因可能会失败，所以尽量设置代理，我上面的示例设置的是淘宝的代理。
